This example is currently set to execute on the development computer with simulated IO.

 To run this example on real hardware, please right-click the FPGA Target in the project explorer window, 
select "Execute On", then "FPGA Target".

You may also need to move this project to a new target. For information about how to do this, please use a web browser to navigate to
 ni.com/info, and use the info code fpgaex.
